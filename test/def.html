Vous êtes sur le site de l'entreprise FICTIVE Microshop. Cette entreprise n'as pour le moment aucun but si ce n'est parler de nous. Il est née d'une blague entre amis. Vous retrouverez sur ce site ce qu'il peut y avoir sur un site d'entreprise classique, même si je me suis laissé la liberté de mettre quelques extravagances.
